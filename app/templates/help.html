Help & Getting Started

Quick Start Guide

  1. Create an Account: Register a new account to start your journey
  2. Browse Challenges: Visit the Challenges page to see all available CTF challenges
  3. Select a Challenge: Start with Easy challenges if you're new to cybersecurity
  4. Read the Description: Each challenge has hints and guidance
  5. Find the Flag: Exploit the vulnerability to discover the flag
  6. Submit Your Flag: Enter the flag in the format FLAG{...} to earn points

Speedrun Mode

Speedrun mode is a competitive feature where you race against time:

  • Select how many flags you want to find (3, 5, 8, or 10)
  • Get a random set of challenges in a separate tenant
  • Timer starts immediately
  • Complete all flags as fast as possible
  • Compete on the speedrun leaderboard

Flag Format

All flags in this lab follow the format:

FLAG{some_unique_identifier_here}

Make sure to submit the complete flag including the curly braces.

Tips for Success

  • Read Carefully: Challenge descriptions often contain important hints
  • Use Browser DevTools: Inspect network requests, cookies, and page source
  • Think Like an Attacker: Consider what an attacker would try
  • Learn from OWASP: Visit the OWASP Top 10 page for detailed explanations
  • Practice Safely: Only use these techniques in this controlled lab environment
